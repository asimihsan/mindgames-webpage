About CrosSequence Program

Introduction

Fill a grid such that cross cells at any location are in sequence. Cross cells are left and right, and, top and bottom of a cell. Sequence is a mathematical sequence like 1,2,3 or 4,6,8 or 1,5,9 etc. There are no duplicate numbers in any row or any column of the grid. Some cells are given and some may be blacked out (not part of a sequence). Puzzle size is 5x5 to 9x9. Each puzzle has a unique solution that can be reached by logic alone without any guesswork.

Program Features

Main features of this program are:
  1. Create new puzzles, grid size 5x5 to 9x9
  2. Create and save puzzles
  3. Autosolve a puzzle with a log of solution steps
  4. Save solution log
  5. Print single or multiple puzzles
  6. Export puzzles in text format
  7. Copy a puzzle to Clipboard
  8. Set display colour schemes
  9. Use one of 20,000 ready made puzzles 2.17Mb

Solving a Puzzle Manually

Normally you would print this puzzle and solve on paper. Puzzle can also be solved on screen.
  1. Navigate through the grid using arrow keys
  2. Enter numbers using number keys
  3. Press Escape to clear a cell
  4. Rightclick on a cell to check validity of two sequences around a number. A green line will indicate a correct while red line will signify an incorrect sequence.
  5. Click verify command button to check if you have solved the puzzle

Auto Solving a Puzzle

Any puzzle created by this program can be solved by the computer. Three options are provided:
  1. Solve Direct : Puzzle is solved in one shot with solution displayed at the end
  2. Solve with a log : Puzzle is solved in one shot but a solution log is maintained giving logic behind the solution. Use Save Solution Log menu to save and print the log in order examine the solution steps at leisure.
  3. Solve with a log and pause at each solution step: Program will display logic behind a solution step and pause. Click on the step message window or press space bar to proceed to next step. Rightclick on the step message window to cancel pause.
  4. Each puzzle has a unique solution which can be reached by logic alone without any guesswork. However, the solution steps taken by a computer solution may not be optimum. Generally a good human solver will get a solution in less steps than the computer.

Printing

The current puzzle and its solution will be printed on an A4 sheet that can be folded in two halves to separate puzzle from its solution. You can also print multiple puzzles from an existing puzzle file.
